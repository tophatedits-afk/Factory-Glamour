# 🏭 Factory Glamour

[![Play Online](https://img.shields.io/badge/Play-Factory_Glamour-2ea44f?style=for-the-badge)](https://tophatedits-afk.github.io/Factory-Glamour/)

Factory Glamour is a browser-based resource processing game where you transform raw materials into valuable manufactured products.

Mine ore batches, process them into resources, smelt them into molten metal, forge ingots, and finally craft finished goods as you build your industrial empire.

> ⚠️ The game is currently in active development.

---

## 🎮 Gameplay

Factory Glamour revolves around a simple production chain:

```
Mine Batch
    ↓
Process Batch
    ↓
Raw Ores
    ↓
Molten Metals
    ↓
Ingots
    ↓
Crafted Products
```

Each stage increases the value of your materials while requiring more processing.

Current resources include:

- 🩶 Iron
- 🟠 Copper
- 🟡 Gold
- 💎 Diamond

## 📦 Production Chain

### ⛏️ Mining
Generate a batch of raw materials.

### 🏭 Processing
Open the batch to receive randomized ores.

Current drop chances:

| Resource | Chance |
|----------|--------:|
| Iron | 60% |
| Copper | 25% |
| Gold | 10% |
| Diamond | 5% |

### 🔥 Smelting

```
10 Raw Ore
      ↓
1 Molten Metal
```

### ⚒️ Forging

```
2 Molten Metal
       ↓
1 Ingot
```

### 🛠️ Crafting

```
1 Iron Ingot   → 2 Crafted Iron
1 Copper Ingot → 3 Crafted Copper
1 Gold Ingot   → 4 Crafted Gold
```

## 💾 Saving

Factory Glamour automatically saves your progress using your browser's LocalStorage.

No account is required.
